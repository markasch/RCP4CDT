# RCP4CDT

Optimal scheduling for cross-facility workflows - a user guide

This is an online user-guide for our optimal scheduling framework, implemented in the [Exa-AtoW](https://numpex.org/exaatow-architectures-and-tools-for-large-scale-workflows/) project of the [NumPEx](https://numpex.org) program.

The online version of the guide can be viewed [here](https://markasch.github.io/RCP4CDT/).
